### Installing

A step by step series of how to download and install.

Make a copy of repository

```
git clone https://github.com/diegoaraujo85/points-on-map-backend.git
```

enter in folder

```
cd points-on-map-backend
```

install dependencies

```
yarn install
```

run the app

```
yarn dev
```
